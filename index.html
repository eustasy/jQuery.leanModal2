jQuery.leanModal2

A pure solution to putting your own HTML and CSS to work as a modal dialogue.

Defaults

$('.leanmodal').leanModal({ top: '15vh', overlayOpacity: 0.7, closeButton: '.js-leanmodal-close', disableCloseOnOverlayClick: false, disableCloseOnEscape: false, fadeTime: 200, });

Default Example

Modified

$('.leanmodal').leanModal({ top: '1em', overlayOpacity: 0.3, closeButton: '.close-it', disableCloseOnOverlayClick: true, disableCloseOnEscape: true, fadeTime: 0, });
